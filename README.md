# My personal dotfiles
```
Usage: ./dotFilesLink.sh [OPTION]
Options:
  -a, --all          Install everything
  -l, --ln           Make symlinks
  -r, --ros          Install ROS
  -v, --vim          Install Vim
  -t, --tmux         Install tmux
  --font             Install Ricty font
  --latex            Install LaTeX and related software
  -c, --cuda         Install CUDA
  -d, --docker       Install Docker

```
